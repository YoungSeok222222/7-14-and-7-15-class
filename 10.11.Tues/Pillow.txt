파이썬에서 이미지를 처리 하기 위해서는 이미지 처리가 가능한 라이브러리가 필요하다.
기존 파이썬 이미지 처리 라이브러리 : Python Imaging Library(PIL)
2011년 부로 기본 PIL 제공이 중단 ( 호환성 문제 )
PIL 후속 프로젝트로 Pillow 등장
즉, Pillow 가 공식 PIL 이라고 생각하면 된다.

PIL 에서 처리 가능한 이미지 작업
 - 픽셀 단위의 조작
 - 마스킹 및 투명도 제어
 - 흐림, 윤곽 보정 다듬어 윤곽 검출 등의 이미지 필터
 - 선명하게, 밝기 보정, 명암 보정, 색 보정 등의 화상 조정
 - 이미지에 텍스트 추가
 - 등등의 이미지 처리 기능 제공

다른 이미지 처리 라이브러리
 - OpenCV ( 컴퓨터 비전 분야 )
 - Scikit-image ( with. Numpy )
 - Matplotlib ( 2D 시각화 툴. but, 이미지 처리도 가능 )
 - 기타
